## 📖 项目简介
该项目为百度智能云针对某智能硬件初创公司提供的“云端多模态实时互动方案”实战复盘。本单页作品集完整拆解了该项目从线下初访、需求映射、架构选型到 POC 测试启动的售前全链路，并系统性地沉淀了将底层技术术语转化为商业与场景语言的核心策略。

## 🎯 核心亮点
* **业务需求分层拆解：** 深入剖析客户的表层询问与深层驱动，精准识别其在“交互延迟”、“BOM 成本控制（融资叙事模型）”以及“数据主权”等维度的核心诉求。
* **“轻端重云”架构选型：** 针对低算力 RTOS 芯片（ESP32）的性能瓶颈，设计并推导基于端侧 SDK 与实时传输网络（RTN）的云端协同架构，实现大模型能力的低成本接入。
* **售前闭环链路沉淀：** 标准化重构从拜访前研判、现场技术场景化翻译，到会后推动测试环境搭建的完整推进节点。
* **高阶信息可视化呈现：** 网页前端采用现代化的 Bento Box（便当盒网格）设计语言，内置流畅的明暗主题（Dark/Light Mode）切换以及可交互的系统数据流向架构图。

## 🛠️ 技术栈
* **前端实现：** HTML5, CSS3 (自定义变量, Flexbox/Grid), 原生 JavaScript
* **设计规范：** 极简主义 (Minimalist), 卡片式网格 (Bento Grid), 双主题自适应
* **排版字体：** Space Grotesk (标题), Noto Sans SC (正文)

## 🚀 核心洞察
> “售前不仅是技术支持，更是客户技术决策链上信任的建立者。短期的诚实与期望管理比长期的失信成本低得多；将技术特性精准翻译成商业决策语言，是解决方案工程师的核心价值所在。”
